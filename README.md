# 🍽️ Restaurant Data Analysis

A data analysis project developed using Python and Google Colab to analyze historical restaurant datasets from Bangalore and Pune.

## 📖 Project Overview

This project focuses on analyzing historical restaurant data to understand pricing, ratings, review counts, and relationships between different attributes. The analysis is performed using Python libraries and visualized with various charts.

## ✨ Features

- Historical Restaurant Data Analysis
- Data Exploration
- Statistical Summary
- Histogram Visualization
- Price vs Rating Analysis
- Pair Plot Visualization
- Correlation Heatmap
- Google Colab Implementation

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📂 Dataset

- Bangalore Restaurants Dataset
- Pune Restaurants Dataset

## 📊 Output

- Data Summary
- Histograms
- Pair Plots
- Correlation Heatmaps
- Price vs Rating Analysis
